## Usage

Be sure to create the database container (I used docker) and correctly configure the ormconfig.json. 
Then...

 ### API
  #$ cd server

    $ yarn
  
    $ yarn typeorm migration:run
    
 ### Start the API
 
    $ yarn dev:server


 ### Web App
  
    $ yarn
    
    $ yarn start
